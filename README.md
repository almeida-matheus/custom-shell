## Installation

#### 1. Clone git repository
```
git clone "https://github.com/almeida-matheus/custom-bash"
```

#### 2. Add permission to execute
```
cd custom-shell/
chmod +x custom-bash.sh
chmod +x custom-zsh.sh
```
#### 3. Execute the script
```
./custom-bash.sh
```
or
```
./custom-zsh.sh
```
